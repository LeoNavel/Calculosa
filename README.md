## Development installation
### Mac OS X
_Note: Homebrew required_
```
$ brew install nodejs
$ brew install Caskroom/cask/electron
$ npm install
```
Then you can run the application
```
$ electron .
```
### Windows
Install NodeJS and add it to path.
```
> npm install
> npm start
```
